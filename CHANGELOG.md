# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2025-12-29

### Added
- **Navigation**: Implemented seamless client-side routing using React Router.
- **Custom Profiles**: Added support for custom player names in both local and online game modes.
- **Real-time Reactions**: Integrated an interactive emoji reaction system for live player expression.

### Changed
- **UI Revamp**: Complete visual overhaul featuring a modern glassmorphic design, glitch effects, and 3D-rendered assets.
- **Voice Chat**: Enhanced voice chat experience with PeerJS, featuring smooth animations and audio visualizations.

## [1.1.0] - 2025-12-28

### Added
- RealTime Voice Chat (WebRTC).

### Fixed
- Performance Improvement.

## [1.0.1] - 2025-12-27

### Changed
- Lock the game state and Dice until required room is full in Online modes.

### Fixed
- Game crash on winning Snake & Ladders in Online mode.
- Turn not switching in Online Ludo.


## [1.0.0] - 2025-12-26
*Initial release.*

# Features
- 3 Classic board games (Tic-Tac-Toe, Snake & Ladders, Ludo).
- Upto 4 player support in Snake & Ladders and Ludo.
- Play with AI.
- Play Online with friends.
- Custom Server URL.
- Free to play.
- No ads.
- No in-app purchases.
